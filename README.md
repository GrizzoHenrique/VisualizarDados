## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de **praticar técnicas de visualização de dados utilizando Python**, aplicando conceitos de **Análise Exploratória de Dados (EDA)** para compreender padrões, tendências e relações entre variáveis em um conjunto de dados.

As visualizações foram construídas utilizando as bibliotecas **:contentReference[oaicite:0]{index=0}** e **:contentReference[oaicite:1]{index=1}**, amplamente utilizadas no ecossistema de **Data Science** para criação de gráficos informativos e interpretáveis.

## 🎯 Objetivos

- Desenvolver habilidades em **visualização de dados**
- Aplicar conceitos de **análise exploratória (EDA)**
- Criar gráficos claros e informativos para interpretação de dados
- Praticar o uso de bibliotecas de visualização do ecossistema Python

## 🛠️ Tecnologias Utilizadas

- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook**

## 📊 Tipos de Visualizações

Durante o projeto foram desenvolvidos diferentes tipos de gráficos para análise dos dados:

- 📈 Histogramas
- 📊 Gráficos de barras

- 📦 Boxplots
